Here is template of react/typescript App.

### Start

1. Clone this repository

```sh
$ cd template
$ yarn
$ yarn start
```
> Project staring on port: 3333. You can set port using APP_PORT in .env root file.

### Build

```sh
$ yarn build
```

### Technologies used: 

1. TypeScript
2. React
3. Babel
4. Webpack
5. Scss
